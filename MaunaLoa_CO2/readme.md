# CO2 levels at Mauna Loa


Background: This data comes from the Mauna Loa Observatory in Hawaii, which has been measuring atmospheric carbon dioxide continuously since 1958.

The Data: Monthly average $CO_2$ concentrations in parts per million (ppm).

More information and metadata can be found [here](https://gml.noaa.gov/ccgg/trends/data.html)

Source: NOAA Global Monitoring Labratory

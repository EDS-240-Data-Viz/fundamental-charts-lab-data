# Data for Fundamental charts lab hackathon

This collection contains 7 datasets for students to  explore, and  visualize with the knowledge they have gained from the first four weeks of EDS 240. 

###  Datasets

| Dataset Name | Source | Key Metrics |
|--------------|--------|-------------|
| [European Vehicle CO2 Emissions](https://www.eea.europa.eu/en/datahub/datahubitem-view/5d252092-d328-40d8-bca2-c0734bd6143b) | EEA | CO2 g/km, Fuel Type, Mass |
| [Global Palm Tree Biodiversity](https://github.com/rfordatascience/tidytuesday/tree/main/data/2025/2025-03-18) | TidyTuesday | Height, Diameter, Altitude |
| [What a Waste: Global Waste Database](https://datacatalog.worldbank.org/search/dataset/0039597/what-a-waste-global-database) | World Bank | Plastic %, GDP, Waste/Capita |
| [Atmospheric CO2 ](https://gml.noaa.gov/ccgg/trends/data.html) | NOAA | Monthly CO2 (ppm) |
| [NA & Greenland Snow Cover Extent](https://www.ncei.noaa.gov/access/monitoring/snow-and-ice-extent/snow-cover/namgnld/12) | NCEI | Million kmÂ² Coverage |
| [Spain Wildfire Dynamics (Joined)](https://gwis.jrc.ec.europa.eu/apps/gwis.statistics/seasonaltrend) | GWIS | Burnt Area, Emissions, Fire Count |
| [Global Energy & Economic Growth](https://github.com/owid/energy-data) | OWID | Renewables, GDP, Consumption |

# Palm Trees

Background: Palm trees (family Arecaceae) are some of the most ecologically and economically important plants on Earth. This dataset, curated for the TidyTuesday project, aggregates observations from botanical gardens, field surveys, and global biodiversity databases.

The Data: The dataset includes thousands of records of palm species found globally. 

More information and metadata can be found [here](https://github.com/rfordatascience/tidytuesday/tree/main/data/2025/2025-03-18)

Source: TidyTuesday

# European Vehicle Emissions 

Background: Maintained by the European Environment Agency (EEA), this dataset tracks Average CO2 emissions per km for new passenger vehicles in Europe. 

The Data: Contains yearly Average CO2 emissions per km from new passenger cars for countries in the EU, Iceland, and Norway. 

More information and metadata can be found [here](https://www.eea.europa.eu/en/datahub/datahubitem-view/5d252092-d328-40d8-bca2-c0734bd6143b)

Source: European Environment Agency
